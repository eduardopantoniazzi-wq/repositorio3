      Importar planilha de controle de pallets

      Envie o arquivo .xlsx atualizado da planilha de controle. O app recalcula automaticamente o estoque disponível e o resumo mensal a partir dele, substituindo os dados de exemplo.

      Como as linhas da planilha são classificadas

      • Saída do estoque + Status Não Retorna → vira estoque disponível para venda (agrupado por transportadora/cliente + estado).
      • Saída do estoque + Status Retorna → ignorada (já está controlada).
      • Entrada no estoque + Motivo COMPRA → conta como custo de reposição, não como pallet em campo.
      • Entrada no estoque + outros motivos (transportadora) → conta como retorno físico.

      Colunas usadas: Tipo de Movimento, Data, Quantidade, Motivo, Estado, Status. As colunas Turno, Observação, Vale Palete e Id Item são ignoradas (Descrição do Item, se existir, é usada só para filtrar linhas que não sejam de pallet).
